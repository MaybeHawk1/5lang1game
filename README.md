# Depenencies
Debian (And any debian based distro):
	   ```sh
	   sudo apt install lua5.4 python3 gcc g++
	   ```\n
Arch (And any arch based distros like endeavour):
	 ```sh
	 sudo pacman -S lua5.4 python3 gcc g++
	 ```\n
Void:
	```sh
	sudo xbps-install -S lua python3 gcc g++
	```\n
